After the grammar and dictionary of Tibetan, we also need some reading materials. For this I have selected this short Tibetan song, which is analyzed in detail by Johan van Manen, the first Dutch Tibetologist.

Dealing with this text is a little more complicated, because often we only have the Tibetan script in view, unlike in the dictionary, which often gave the transcription right away. We still use the same encoding scheme for Tibetan (བོད་སྐད་) script. This book, too, assumes a working knowledge of Devanagari (देवनागरी)

Unlike the dictionary, I've attempted to OCR the Tibetan script. Unfortunately, the quality of Tibetan OCR is not yet as good as for Latin script, so expect a lot of mistakes. Note also, that the quality of OCR for English is a little bit less than we are used to, because the OCR software that supports Tibetan isn't that good on English either (and does sometimes mix up the scripts -- I also tried supporting Devanagari OCR at the same time, but that had devastating results).

The original scans are in the Internet Archive. A few missing pages have been supplied from Google Books.

Diacritics

This book follows the Latin transliteration scheme of Jäschke, except that it does away with some of it's more complicated aspects. The turned comma for aspirated letters is simply typed after, not on top of, the aspirated consonants, and we can thus simply type it as k`.

Tibetan Script

The below table is for quick reference only; please take care to read the full transcription instructions if you are not already familiar with Tibetan script and Wylie encoding.

If you are unsure, you can always try things in the Online Tibetan conversion tool.

Tibetan is to be typed between [BO: and ]; when you encounter a new-line in Tibetan, please end the line with backslash (\), if needed preceded with a space for the tsheg. (Otherwise, the space representing the tsheg at the end of a line will be lost.)

You can include asterisks in the Tibetan text as [[*]] (one open bracket to go to Latin script mode, the other is a literal bracket, following our convention with regard to note markers; second closing bracket goes back to Tibetan text mode.)
